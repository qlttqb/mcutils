=====================================================
  MINECRAFT ENCHANTMENT CALCULATOR — CUSTOM ICONS
=====================================================

Drop your icon image files into THIS folder (icons/).
The calculator will automatically use them instead of
the default emoji icons.

-----------------------------------------------------
HOW TO NAME YOUR FILES
-----------------------------------------------------

Name each file after the item's ID listed below.
Supported formats: PNG, JPG, JPEG, GIF, WEBP, SVG

Examples:
  sword.png
  pickaxe.png
  chestplate.png
  bow.jpg
  elytra.webp

-----------------------------------------------------
FULL LIST OF ITEM IDs
-----------------------------------------------------

ARMOUR:
  helmet

  leggings
  boots
  turtle_shell
  elytra

WEAPONS:
  sword

MISC:
  carrot_stick
  warped_stick
  spear
  pumpkin

-----------------------------------------------------
TIPS
-----------------------------------------------------

- Recommended size: 64x64 px or larger (square works best)
- The calculator shows icons at 40x40 px on item cards
- You can use Minecraft texture pack sprites directly!
- If a file is missing, the emoji fallback shows instead
- You can mix and match — only replace the ones you want
